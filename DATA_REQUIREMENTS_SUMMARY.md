## 📊 A股 (asset_type = "STOCK_A")

### 需要获取的数据

```
资产代码: sh600519
↓
AkShare API: ak.stock_zh_a_spot_em()
↓
返回的关键字段:
  代码 ............... 600519 (项目格式: sh600519)
  名称 ............... 贵州茅台
  最新价 ............. 1820.50 ✅ 用于持仓价值计算
  昨收 ............... 1815.30
  涨跌幅 ............. 0.28 ✅ 用于CHANGE_PCT告警
  涨跌额 ............. 5.20
  成交量 ............. 1250000 ✅ 用于VOLUME告警
```

### 数据获取特点

```
✅ 一次 API 调用拉取全市场 (~5000 只股票)
✅ 耗时: 1-3 分钟 (AkShare 从多个源聚合)
✅ 我们缓存 60 秒
✅ 同期间多个持仓查询共用这一次 API 调用
```

## 📈 基金 (asset_type = "FUND")

### 需要获取的数据 - 3 种基金类型

#### 🏢 类型 1: ETF (交易所交易基金)

```
代码示例: 510300 (沪深 300 ETF)
↓
自动识别: 6位数代码 5xxxxx 或 1xxxxx → ETF
↓
AkShare API: ak.fund_etf_spot_em()
↓
返回的关键字段:
  代码 ............... 510300
  名称 ............... 华夏沪深 300
  最新价 ............. 3.850 ✅ 实时市价
  昨收 ............... 3.845
  涨跌幅 ............. 0.13 ✅ CHANGE_PCT 告警
  成交量 ............. 45600000 ✅ VOLUME 告警
```

#### 🏛️ 类型 2: LOF (上市开放式基金)

```
代码示例: 166009 (中欧新趋势)
↓
自动识别: 其他交易所交易基金 → LOF
↓
AkShare API: ak.fund_lof_spot_em()
↓
返回的数据结构: 与 ETF 相同
  最新价 ............. (实时市价)
  涨跌幅 ............. (CHANGE_PCT)
  成交量 ............. (VOLUME)
```

#### 🤝 类型 3: 开放式基金

```
代码示例: 005827 (南方养老 2035)
↓
自动识别: 其他 6 位数代码 → 开放式基金
↓
AkShare API: ak.fund_open_fund_daily_em()
↓
返回的关键字段:
  基金代码 ........... 005827
  基金简称 ........... 南方养老 2035
  2026-04-03-单位净值 .. 1.4856 ✅ T+1 净值（非实时！）
  日增长率 ........... 0.05 ✅ CHANGE_PCT 告警
  (无成交量) ......... None ❌ VOLUME 告警不可用

⚠️ 关键限制:
  • 仅有 T+1 净值，不是实时市价
  • 无成交量数据
  • 列名动态生成（日期前缀）
```

### 基金数据获取特点

```
✅ 按类型自动识别（根据 6 位代码模式）
✅ 批量拉取全市场，缓存 60 秒
✅ ETF/LOF 有实时市价 + 成交量
⚠️  开放式基金仅 T+1 净值，无成交量
```

## 💰 黄金 (asset_type = "GOLD_SPOT")

### 需要获取的数据

```
品种代码: Au99.99 (上海黄金标准品)
↓
优先使用: ak.spot_quotations_sge() (实时 tick)
↓
关键字段:
  品种 ............... Au99.99
  现价 ............... 480.5 ✅ 单位: 元/克
  时间 ............... 2026-04-03 14:30:00

⚠️ 如果实时 API 返回空 (非交易时间):
  ↓
  自动 Fallback: ak.spot_hist_sge()
  ↓
  取最后一行历史数据
    date ............ 2026-04-02
    close ........... 479.2 ✅ 用户无感知
    open ............ 477.8
    high ............ 481.0
    low ............. 476.5
```

### 黄金数据获取特点

```
✅ 实时 API 仅交易时间有效
✅ 非交易时间自动使用历史最后一行
✅ 历史数据缓存 300 秒
⚠️ SGE 不公布成交量 → VOLUME 告警不可用
```

   │  (A股全市场，缓存 60s)              │

   └─────────────────────────────────────┘
   ┌─────────────────────────────────────┐
   │  fund._get_etf_data_cached()        │
   │  (ETF全市场，缓存 60s)              │
   │  → 510300: 最新价 3.850             │
   ├─────────────────────────────────────┤
   │  fund._get_open_fund_data_cached()  │
   │  (开放式全市场，缓存 60s)           │
   │  → 005827: 净值 1.4856              │
   └─────────────────────────────────────┘
   ┌─────────────────────────────────────┐
   │  gold.get_quote()                   │
   │  (黄金实时，缓存历史 300s)          │
   │  → Au99.99: 现价 480.5              │
   └─────────────────────────────────────┘
   ↓
   返回字典:
   {
     ('STOCK_A', 'sh600519'): QuoteData(current_price=1820.50, ...),
     ('FUND', '510300'): QuoteData(current_price=3.850, ...),
     ('FUND', '005827'): QuoteData(current_price=1.4856, ...),

   }

   用于计算持仓总价值、告警判断

## 🎯 API 调用示例

### 导入

```python
from app.data_fetcher import get_quote, get_quote_batch
```

### A股

```python
quote = get_quote('STOCK_A', 'sh600519')
print(f"贵州茅台当前价: ¥{quote.current_price}")  # ¥1820.50
print(f"涨幅: {quote.change_pct}%")                # 0.28%
```

### 基金

```python
# ETF
quote = get_quote('FUND', '510300')
print(f"沪深 300 当前价: ¥{quote.current_price}")

# 开放式
quote = get_quote('FUND', '005827')
print(f"南方养老当前净值: ¥{quote.current_price}")  # T+1
```

### 黄金

```python
quote = get_quote('GOLD_SPOT', 'Au99.99')
print(f"黄金现价: ¥{quote.current_price}/克")  # ¥480.5/克
```

### 批量查询 (推荐)

```python
positions = [
    ('STOCK_A', 'sh600519'),
    ('STOCK_A', 'sh000001'),
    ('FUND', '510300'),
    ('FUND', '005827'),
    ('GOLD_SPOT', 'Au99.99'),
]

quotes = get_quote_batch(positions)

# 计算总资产价值
total_value = 0
for (asset_type, symbol), quote in quotes.items():
    qty = database.get_holding_quantity(asset_type, symbol)
    total_value += quote.current_price * qty

print(f"总资产: ¥{total_value}")
```
